## Overview

A modern, responsive marketing intelligence dashboard with an AI-like prompt playground for campaign analysis.

## Features

- Campaign performance dashboard (CTR, conversions, impressions)
- Filters: status, sort by CTR / conversions, search
- AI Prompt Playground (keyword-based intent parsing)
- Zustand for global state management
- Recharts for data visualization
- Fully responsive (mobile + desktop)
- SEO + performance optimized

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Zustand
- Recharts

## Architecture

- Modular component structure
- Global store for campaign state
- Utility-based prompt parsing
- Performance optimized using useMemo

## Setup

```bash
git clone <repo>
cd ai-vibe-dashboard
npm install
npm run dev
```
